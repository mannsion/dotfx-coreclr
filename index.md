# .NET 10 CLR API Reference

This site is generated from the .NET 10 reference assemblies installed with the local SDK. It gives you a browsable API reference for the public surface area exposed by `Microsoft.NETCore.App.Ref`.

## Start Here

- [Runtime API Map](runtime-map.md): curated links into the most useful CLR and base-class-library APIs.
- [Generated API Reference](api/index.md): the full DocFX-generated namespace and type browser.
- [Microsoft .NET API Browser](https://learn.microsoft.com/en-us/dotnet/api/): upstream docs with conceptual articles, examples, and remarks.

## What This Includes

- Public .NET 10 reference assemblies from `Microsoft.NETCore.App.Ref`.
- XML documentation included with the installed reference pack when available.
- Types under namespaces such as `System`, `System.Reflection`, `System.Runtime`, `System.Threading`, `System.IO`, `System.Net`, and related assemblies.

## What This Does Not Include

This is not a documentation build of the native CoreCLR implementation. It will not show the C++ runtime source, JIT internals, GC implementation internals, VM object layout code, or runtime design notes.

For runtime implementation internals, use the `dotnet/runtime` repository source and docs instead.
